⚙️ Technical Setup Guide

📋 Prerequisites

  • Windows 11 (or Windows 10)
  • Python 3.13 (or 3.10+)
  • DHAN trading account(s)
  • API access enabled on DHAN
  • Minimum ₹25,000 capital

1. Clone Repository

git clone https://github.com/yourusername/jiraiya-ai-trader.git cd jiraiya-ai-trader

2. Create Virtual Environment

python -m venv venv venv\Scripts\activate # Windows

3. Install Dependencies

pip install -r requirements.txt

4. Configure Environment (.env file)

# Data Source (Profile 1) DHAN_CLIENT_ID_1='your_client_id_1' DHAN_TOKEN_1='your_access_token_1' # Trading Account (Profile 3) DHAN_CLIENT_ID_3='your_client_id_3' DHAN_TOKEN_3='your_access_token_3' # Specify data profile DHAN_DATA_PROFILE_ID='1' # Telegram (Optional) TELEGRAM_BOT_TOKEN='your_bot_token' TELEGRAM_CHAT_ID='your_chat_id' TELEGRAM_ENABLED=true

5. Get DHAN API Credentials

  1. Login to https://dhan.co
  2. Go to Settings → API Access
  3. Click "Generate Access Token"
  4. Copy Client ID and Access Token
  5. Paste into .env file
  6. Repeat for all profiles you want to use

6. Configure Trading Parameters (autobot_config.json)

{ "selected_symbol": "NIFTY", "max_amount_per_trade": 20000.0, "candle_minutes": 1, "risk_mode": "inr", "risk_tp_value": 10000.0, "risk_sl_value": 1000.0, "trailing_stop_enabled": true, "trail_trigger_value": 300.0, "trail_gap_value": 50.0, "trail_step_value": 20.0, "max_profit_per_day": 100000.0, "max_sl_per_day": 5000.0, "max_trades_per_day": 28 }

7. Run Pre-Production Tests

python pre_production_test.py

Ensure all tests pass before going live.

8. Start Application

python gui_app.py

9. Select Profile & Mode

  • In GUI, select "Profile 3" (or your trading profile)
  • Start in DRYRUN mode for paper trading
  • Monitor for a few hours
  • Switch to LIVE mode when ready
